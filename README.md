# Калькулятор дозы инсулина — инструкция по установке

Это отдельное веб-приложение (не артефакт Claude). Фото Dexcom и этикеток будут
работать надёжно, потому что вызовы ИИ идут через собственный сервер, а не через
ограничения мобильного приложения Claude.

## Что нужно перед началом

1. Аккаунт на **vercel.com** (бесплатно, регистрация через Google/GitHub — 1 минута)
2. Один API-ключ Anthropic (получить на **console.anthropic.com** → API Keys → Create Key)
   Понадобится небольшой баланс на счету (несколько $ хватит на очень долгое использование,
   т.к. каждый запрос стоит доли цента)

## Установка (10 минут, один раз)

### Шаг 1: Загрузить проект на Vercel

Самый простой способ — через сайт Vercel, без использования терминала:

1. Зайдите на **vercel.com** → войдите через GitHub (если нет аккаунта GitHub — создайте на github.com, тоже бесплатно и быстро)
2. На GitHub создайте новый репозиторий (кнопка "New repository"), например `insulin-calculator`
3. Загрузите туда все файлы из этой папки (просто перетащите файлы на страницу репозитория через "Add file" → "Upload files")
4. Вернитесь на Vercel → "Add New" → "Project" → выберите ваш репозиторий `insulin-calculator`
5. Vercel сам определит что это Vite-проект. Нажмите "Deploy"

### Шаг 2: Добавить API-ключ (важно — без этого фото не будут работать)

1. В панели Vercel откройте ваш проект → **Settings** → **Environment Variables**
2. Добавьте переменную:
   - Name: `ANTHROPIC_API_KEY`
   - Value: ваш ключ с console.anthropic.com (начинается с `sk-ant-...`)
3. Нажмите **Save**
4. Перейдите в **Deployments** → на последнем деплое нажмите "..." → **Redeploy**
   (нужно чтобы новая переменная подхватилась)

### Шаг 3: Добавить на экран iPhone

1. Vercel даст вам ссылку вида `https://insulin-calculator-xxxx.vercel.app`
2. Откройте эту ссылку в **Safari** на iPhone
3. Кнопка "Поделиться" → **"На экран «Домой»"**
4. Готово — иконка на рабочем столе, фото будут работать стабильно

## Если что-то не получится

Самый частый затык — шаг с GitHub. Если неудобно возиться с загрузкой файлов
через веб-интерфейс, альтернативный (тоже бесплатный) вариант — попросить
знакомого с опытом немного помочь с загрузкой через `git`, это займёт у него
5 минут: `git init`, `git add .`, `git commit`, `git push`.

## Что изменить в будущем

Все настройки (УК по времени, ФЧ, коэффициент БЖП, доза Тресибы) находятся
в файле `src/App.jsx` в самом начале — можно попросить Claude отредактировать
любое значение и просто загрузить обновлённый файл заново на GitHub
(Vercel обновит сайт автоматически за минуту).
